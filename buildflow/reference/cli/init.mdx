The init command will create a new BuildFlow application in your currenct directory.

When you run `buildflow init` you will be prompted to fill out some basic information about your application. This informattion will be used to create the basic directory structure and layout of your application.

Options:
- `--app` - The name of the application. This will be used to name the application directory and the application in the `buildflow.yaml` file.
- `--directory` - The directory to initialize. Defaults to the current directory.

Run `buildflow init --help` to see available options:

```
$ buildflow init --help
```
